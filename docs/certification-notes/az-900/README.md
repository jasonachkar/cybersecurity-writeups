Welcome to the **Microsoft AZ-900: Azure Fundamentals** study guide. This course is divided into dedicated sub-sections covering the core topics of the official exam curriculum in technical depth.

## Study Guide Directory

* **[Domain 1: Cloud Concepts](domain-1-concepts.md)**
    * Learn about cloud computing benefits (high availability, scalability, elasticity, agility).
    * Understand operational financial models (CapEx vs. OpEx).
    * Analyze the Shared Responsibility Model and Cloud service archetypes (IaaS, PaaS, SaaS).
* **[Domain 2: Core Azure Architecture & Services](domain-2-architecture-services.md)**
    * Learn about regions, availability zones, management groups, and subscriptions.
    * Deep dive into VMs, ACI, AKS, serverless Azure Functions, and VNets.
    * Review storage architectures (Blobs, Files, Disks) and database engines.
* **[Domain 3: Management, Security, and Governance](domain-3-management-governance.md)**
    * Master Entra ID authentication and Conditional Access.
    * Detail security policies, RBAC roles, resource locks, and cost management.

## About the AZ-900 Exam
The Microsoft Certified: Azure Fundamentals certification validates foundational knowledge of cloud services and how those services are provided with Microsoft Azure. 

### Exam Outline
* **Describe Cloud Concepts:** 25–30%
* **Describe Azure Architecture and Services:** 35–40%
* **Describe Azure Management and Governance:** 30–35%
